1. Components
- Configuration
- Logging

2. Interactions
- Configuration → Logging : RAIILogStream.stream

3. Execution flow
- The Configuration component’s PropertyMapper executes several property operations (add, addEnum, addEigen, addVector, getChild, mandatoryCheck). 
- Each of these operations issues a log via CPSLOG_* macros, which construct a RAIILogStream and invoke its stream method in the Logging component. 
- Successful property reads (e.g., in add, addEnum) log trace messages, while missing mandatory keys trigger error logs (e.g., in getChild and mandatoryCheck). 
- The trace captures this as a series of Configuration → Logging calls to RAIILogStream.stream for each of the PropertyMapper methods executed.
